# InvestMind

iOS приложение для инвестиций, созданное с использованием SwiftUI.

## Описание

InvestMind — это мобильное приложение для управления инвестиционным портфелем, просмотра информации об акциях и обучения инвестициям. Приложение предоставляет пользователям удобный интерфейс для отслеживания активов, изучения рынка и принятия инвестиционных решений.

## Технологии

- **SwiftUI** — фреймворк для создания пользовательского интерфейса
- **iOS 18.5+** — минимальная версия iOS
- **Xcode 16.4+** — среда разработки

## Структура проекта

### Экраны приложения

- **Splash Screen** — экран загрузки с логотипом InvestMind
- **Onboarding** — вводный экран с информацией о приложении (2 страницы)
- **Survey** — опрос для настройки приложения (2 страницы с выбором опций)
- **Authentication** — экран авторизации (номер телефона и пароль)
- **Registration** — экран регистрации (номер телефона, пароль, подтверждение пароля)
- **Main Tabs** — главный экран с вкладками (акции, портфель, новости, профиль)

### Компоненты

- `AppTheme` — централизованная система дизайна (цвета, шрифты, отступы)
- `FilledButton` — кнопка с заливкой
- `SocialLoginButton` — кнопка для входа через социальные сети
- `InvestTextField` — кастомное поле ввода с валидацией
- `GuideCard` — карточка для гайда

### Шрифты

- **Philosopher** — используется для логотипа "InvestMind" на всех экранах
- **SF Pro** — системный шрифт для всего остального текста

## Статус разработки

Приложение находится в стадии разработки UI. В настоящее время реализованы:

- ✅ Интеграция кастомных шрифтов
- ⏳ API интеграция (в планах)

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте `InvestMind.xcodeproj` в Xcode
3. Выберите симулятор или устройство
4. Нажмите Run (⌘R)

## Особенности

- Адаптивный дизайн
- Валидация форм с анимированными сообщениями об ошибках
- Кастомные цвета и градиенты
- Поддержка темной темы (черный фон)
- Маски ввода для номеров телефонов
- Переключение видимости пароля
